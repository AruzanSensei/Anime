# Cara Download Anime

Panduan lengkap cara menggunakan aplikasi untuk download anime.

## ğŸ¯ Langkah-langkah Download

### 1. Cari Anime
- Ketik judul anime di search bar (contoh: "Naruto", "One Piece")
- Klik tombol **"Cari"** atau tekan Enter
- Tunggu hasil pencarian muncul

### 2. Pilih Anime
- Klik pada anime yang ingin kamu download dari tabel hasil
- Episode akan dimuat otomatis

### 3. Pilih Kualitas
- Pilih resolusi yang diinginkan dari dropdown **"Kualitas"**
- Pilihan tersedia:
  - **Highest MP4** (Recommended) - Kualitas terbaik MP4
  - **Highest MKV** - Kualitas terbaik MKV
  - MP4 720p, 480p, 360p
  - MKV 1080p, 720p, 480p

### 4. Pilih Episode
Kamu punya 2 opsi:

**Opsi A: Unduh Semua**
- Klik tombol **"Unduh Semua"**
- Semua episode akan masuk antrian download

**Opsi B: Unduh yang Dipilih**
- Masukkan nomor episode di kolom **"Episode yang ingin diunduh"**
- Format:
  - Range: `1-5` (episode 1 sampai 5)
  - Spesifik: `1,3,5` (episode 1, 3, dan 5)
  - Kombinasi: `1-3,5,7-10`
- Klik tombol **"Unduh yang Dipilih"**

### 5. Download File

**PENTING:** Cara download berubah untuk menghindari error hotlink dari Pixeldrain.

Setelah klik tombol download:

1. **Tab baru akan terbuka** menuju halaman Pixeldrain
2. Di halaman Pixeldrain, kamu akan lihat:
   - Preview video (jika browser support)
   - Informasi file (nama, ukuran, dll)
   - Tombol **"Download"** berwarna hijau
3. **Klik tombol "Download"** di halaman Pixeldrain
4. File akan mulai download ke folder Download HP/PC kamu

**Catatan:**
- Aplikasi akan membuka tab baru secara otomatis untuk setiap episode
- Jika download banyak episode, akan ada banyak tab yang terbuka
- Tutup tab setelah download selesai untuk menghemat memori

## â“ Troubleshooting

### Tab Tidak Terbuka
**Problem:** Tombol "Buka Link" tidak membuka tab baru

**Solusi:**
1. Cek browser tidak memblokir pop-up
2. Klik manual tombol "Buka Link"
3. Izinkan pop-up untuk website ini

### Error "Hotlink Detected"
**Problem:** Muncul error hotlink di Pixeldrain

**Solusi:**
- âœ… Sudah diperbaiki! Sekarang kita buka halaman Pixeldrain langsung
- Klik tombol Download di halaman Pixeldrain

### Download Tidak Mulai
**Problem:** Sudah klik Download tapi tidak ada yang terjadi

**Solusi:**
1. Cek koneksi internet
2. Coba browser lain
3. Clear cache browser
4. Disable ad blocker (jika ada)

### File Tidak Ditemukan di Download
**Problem:** Download selesai tapi file tidak ada

**Solusi:**
1. Cek folder Downloads di File Manager
2. Cari dengan nama file (biasanya nama episode)
3. Cek storage permission browser (mobile)
